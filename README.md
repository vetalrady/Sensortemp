# SensorTemp PWA

This repository is now structured as a Vite + React Progressive Web App.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## GitHub Actions

- `.github/workflows/ci.yml` builds on push and pull requests.
- `.github/workflows/pages.yml` builds and deploys to GitHub Pages from `main` or `master`.

## PWA features

- App manifest and service worker are generated by `vite-plugin-pwa`.
- Assets are cached for installable/offline behavior.
- Sensor icon assets are in `public/`.
