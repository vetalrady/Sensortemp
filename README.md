# SensorTemp

A React + Vite Progressive Web App (PWA) for viewing SensorPush temperature and humidity data.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## PWA features

- Installable app manifest (`name`, SVG icon, standalone display mode).
- Service worker generated by `vite-plugin-pwa` for offline caching.
- Automatic service worker update checks on app load.

## GitHub Actions

Two workflows are included:

1. **Build** (`.github/workflows/build.yml`)
   - Runs on all pushes and pull requests.
   - Installs dependencies and runs `npm run build`.

2. **Deploy to GitHub Pages** (`.github/workflows/deploy-pages.yml`)
   - Runs on pushes to `main`.
   - Builds the app and deploys `dist/` to GitHub Pages.

To enable deployment, make sure Pages is configured in repository settings to use **GitHub Actions** as the source.
